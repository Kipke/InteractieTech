##### RGB Light meaning

State color led

- Standby: off
- Unknown: white
- Cleaning: green
- Menu: blue
- Number One: yellow
- Number Two: red
- Triggered
	- yellow flashing: One shot remaining
	- red flashing: Two shots remaining
